# v0.6.2 — end-of-match panel + toolbar polish

## What's new

### End-of-match panel (replaces "X wins!" card)

When the match ends, the game screen now shows a final-score panel with
three buttons instead of jumping straight out to the menu:

- **History** — per-leg log of every throw, grouped by leg. Each leg
  shows a header (with the winner highlighted) and a 2-column list of
  the two players' turns for that leg.
- **Stats** — per-player summary table for this match only: legs won,
  3-dart avg, first-9 avg, high checkout, 180s, 140+, 100+, checkout %,
  busts, best leg.
- **Finish** — records the result and returns to the Dashboard
  (or the bracket/league in matchMode).

The panel sits in the calcHost slot, so the toolbar, player cards, and
shared history strip stay visible — the user sees the result in
context instead of having the form jump away.

### Toolbar polish

- The live match status (e.g. `2:1 (1:0)`) is now centered in the
  toolbar bar, between the "Game: 501 · DO" text on the left and the
  fullscreen/exit icons on the right.
- Font size matches the "DO" value (`2.4em` ≈ 34.6px) so the status
  reads as a centrepiece, not a footnote.
- The "Game:" label is hidden; only the "501 · DO" value shows.
- "Sets:" and "Legs:" shortened to "S:" and "L:" to keep the bar compact.
