## Gin's Online Dart's Scoring System v0.3.0

Major: dedicated `players` IDB store. Replaces v1.3.0.

### What's new

- **New `players` IDB store**, distinct from `users` (admin accounts).
  Players don't log in; they just have identity fields.
  Fields: `firstName`, `surname`, `middleName`, `town`, `club`,
  `regNumber` (format `CLUBCODE#NNNNNN`), `regAuthority`.
- **Bulk import** — paste a list of players (CSV or whitespace
  separated). Auto-matches existing players by reg# or name.
- **Auto-allocate registration numbers** — when bulk-creating
  players, if the line has a known club name with a known
  shorthand, the next serial is auto-assigned. Otherwise the
  user is prompted for a new home-club shorthand.
- **Home-club filter** on the player picker.

### Migration
`DB_VERSION` bumped from 2 to 3. New `players` store created
on first open. Existing `users` (admin) untouched.

### Tests
88/88 unit + 10/10 viewports pass.
