# v0.3.2 — Custom Cursor + Drive-Synced Settings + Inline Help

**Release date:** 2026-06-26

## Highlights

### Custom Cursor

The app now has a visible, theme-matched pointer:

- **Dart** — dark body, white outline, bullseye-red tip (default)
- **Crosshair / target** — white ring + red center dot
- **Default** — fall back to the system pointer

Five sizes:

| Size | Pixels |
|------|--------|
| Auto | picks 24/28/32/40/48 based on viewport |
| Small | 20 |
| Medium | 28 |
| Large | 40 |
| XL | 56 |

Auto uses the smaller of viewport width/height so phones, tablets,
and desktops all get a comfortable pointer.

### Settings in the Header

A new gear icon sits in the top-right header, to the right of the
sign-out icon. Tap it to open a quick "Settings" modal with three
segmented rows: **Style**, **Size**, and **Help icons**.

### Saved Locally and Synced to Google Drive

- Not signed in: settings are stored in IndexedDB and persist across
  reloads.
- Signed in with Google: cursor settings upload to
  `gins-darts/settings/cursor.json` and help-icon visibility uploads
  to `gins-darts/settings/ui.json` in your hidden `appDataFolder`.
  Next sign-in on another device restores both settings automatically.

### Inline Help Icons in New Competition Form

The "New competition" form now shows a small ⓘ icon next to every
section heading and option row:

- Name, Metadata, Type, Participant Format, Elimination Format,
  Game settings
- Individual options: Seeding, Number of groups, Players advancing,
  Number of teams, Starting score, Double-out, Legs to win,
  Max darts per leg, Rounds, Cut-throat

The previously visible "About this format" / "About this game" panels
are now hidden behind the help icons, keeping the form compact while
still offering full explanations on demand.

Users can show or hide all help icons from the header settings modal.

## Backwards Compatibility

- No database migration needed; new settings use the existing
  `settings` IndexedDB store.
- Drive file layout uses a new `gins-darts/` prefix under
  `appDataFolder` for app settings. Existing competition / match /
  history files are untouched.
- The old toolbar ✕ Quit button is removed; use the calculator ⏻ Exit
  button to leave a game.
